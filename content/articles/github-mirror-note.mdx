# GitHubミラー公開のお知らせ

検索向けの公開面として、`github.com` 上にも同じ記事内容を読める GitHub ミラーを用意しました。

- GitHub ミラー: [sekihan02/sample-ripo](https://github.com/sekihan02/sample-ripo)

このミラーでは、現在公開している記事本文を Markdown としてまとめて読めます。

## このミラーでできること

- `github.com` 上で記事一覧を開く
- そのまま記事本文を読む
- 元の公開サイトへのリンクから元記事へ移動する

## 補足

GitHub ミラーには、記事内チャットのようなサイト固有機能は含めていません。

チャットや通常の閲覧は、これまでどおりサイト本体を利用してください.
